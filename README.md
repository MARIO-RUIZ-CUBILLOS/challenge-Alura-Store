# challenge-Alura-Store
Se implementa estudio de las tiendas de don juan con la finalidad informarle cual de sus tiendas tiene menos ganancias.
esto es con la finalidad de vender dicha tienda y hacer otro negocio.
en este archivo se enceontrara 10 puntos importantes:
1.-Analisis de facturacion
2. Ventas por categoria
3. Calificacion promedio de la tienda
4. Productoas mas o menos venddidos
5. Promedio de costos de envios
6. Graficos varios 
7. Informe --- este punto es el mas importante el cual se explica los datos y como se estan comportando sus tiendas.
8. Trabajo opcional.
